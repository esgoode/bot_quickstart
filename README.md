# Basic Bot (Based off Quickstart and Waterfall)

## To Run:

1. Install Node.js
2. Clone repo
3. Run npm install
4. Run node app.js
5. [Install Chat emulator 3.5.36](https://github.com/Microsoft/BotFramework-Emulator/releases)
6. Open Chat Emulator and Enter: http://localhost:3978/api/messages
7. Begin Chat
